金成秀
金孟镇、李彩英的长子

1987年3月21日出生于大邱。在大邱生活了27年，后来为了就业来到了首尔，现在是首尔(?)人。 从小就很喜欢玩电脑，现在在IT行业靠电脑吃饭。一直想和漂亮的大邱女人结婚，终于办到了。








丁世美
丁旭、刘贤淑的长女

1988年12月7日出生于大邱。一直在大邱生活，后来因为大学开始了在首尔的独自生活，并持续了10多年。青春期时期喜欢偶像组合-神话，在做Fan Art时自学的Photoshop现在仍在公司中使用。小时候喜欢像男偶像一样的美男子类型。但是随着年龄的增长，现在喜欢稳重的男人。现在能与这样的男人共度一生，真的是太好了。

我们 的故事

08
2014

一起照的第一张照片

成秀在naver地图组工作，世美在line工作的时候，在同一个项目中作为各自小组的老幺工作，在项目的结束会餐中第一次见面。在那里连对视都不敢，但在组长们的折腾下，最后单独见面了。可能是因为各自在外地工作时见到老乡(?)而产生的莫名的亲切感吗? 一开始就互相产生了好感，第三次见面时就决定交往了。在青涩的社会新人时期，都没想到那份姻缘会延续5年多，甚至会结为夫妻。

2014

非常美丽的秋季济州岛。想在秋天再去一次。

06
2018

香港旅行

一起去的香港。在中国代表IT企业腾讯工作的世美长期出差到深圳总部时，成秀为了去见世美飞到了这里。

11
2018

两对情侣

和经成秀和世美介绍而结婚的朋友们一起。两人最亲密的朋友结为了夫妻，感到非常新奇和高兴。

01
2019

滑雪场

两个人都喜欢运动。成秀喜欢足球和游泳，世美喜欢瑜伽和游泳。所以一到周末就一起去游泳池。从去年冬天开始，成秀开始教世美滑雪板。因为世美非常讨厌寒冷人，所以一直固执地坚持到春天在去，但还是不得不在寒冷的冬天跟着走，结果比想象中有趣得多。今年两人都买了季票。打算从巴厘岛度完蜜月回来后，马上去滑雪。

05
2019

美国西部旅行，大峡谷

两个人都喜欢旅行。两个人一起去也好，和各自的同伴分开去也好，度假地也好，旅游景点也很好。即使是在百忙之中离开，在旅行地的时间好像流逝地更慢。即使在结婚后忙于生计，也想经常和珍贵的朋友和家人一起去旅行。

05
2019

在谷歌会议中心

两个人都从事IT行业。成秀是开发者，世美是设计师。听到这句话，很多人都说"两个人什么都可以做出来"，但是这份手机请柬是这5年来的第一个。虽然不是同一种职业，但在同一行业工作觉得很好。因为虽然彼此有共同语言，但却不能对彼此的事情干涉。

06
2019

两个人的取向

两个人都是漫威电影迷，都喜欢野菜拌饭。虽然不直接养猫，但喜欢看猫。虽然大家可能会认为我们俩是天生一对，但也许是我们在这一段长久的时间里，彼此互相适应并改变。即使成为夫妻，也依然会是两个不同的人。但是我们会以互相爱护的心去适应人生的每个角落，逐渐成为天生生一对。希望能像以前一样，寻找一起能享受的东西，并和睦地逐渐老去。

09
2019

朋友的婚礼

在前面说的两人的好朋友结婚的婚礼上唱了祝歌。虽然唱得不好，但是留下了美好的回忆(..?)。

09
2019

在摄影棚拍摄时

穿着礼裙和西装在摄影棚拍摄，才真正感觉到"我们是真的要结婚了"。

我们就这样要结婚了。
